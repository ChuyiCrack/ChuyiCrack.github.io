// what it is

The project

Staam is a desktop application inspired by Steam, developed in Java using Swing for the graphical user interface. It allows users to create accounts, authenticate securely, and manage a digital game library through an intuitive desktop experience. The project was developed over three months as part of a collaborative software engineering project.

As the team leader, I coordinated task distribution, managed feature integration, and designed the application's relational SQL database. One of the main technical challenges was integrating independently developed modules into a cohesive application while maintaining a clean architecture and ensuring reliable communication between the user interface, business logic, and database.

The project strengthened my experience in object-oriented programming, database design, Git-based collaborative development, and leading a software team. It also reinforced the importance of planning, code organization, and effective teamwork when building medium-sized desktop applications.
